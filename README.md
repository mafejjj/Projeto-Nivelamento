# Projeto-Nivelamento

Landing page criado para ajudar no combate ao descarte inadequado de resíduos em áreas urbanas. A aplicação permite que cidadãos solicitem coletas, denunciem locais de descarte irregular e encontrem pontos corretos para descarte de materiais.

# Sobre o Projeto

Grandes centros urbanos enfrentam diversos problemas relacionados ao descarte incorreto de resíduos. Materiais como móveis, eletrodomésticos, entulhos, óleo de cozinha, eletrônicos e recicláveis muitas vezes são abandonados em locais impróprios, causando impactos ambientais, riscos à saúde pública e prejuízos ao espaço urbano.

# Funcionalidades
- Cadastro de solicitação de coleta
Permite que o usuário solicite a retirada de resíduos.

- Cadastro de denúncias
Possibilidade de registrar denúncias de descarte irregular.

- Upload de imagens
O usuário pode anexar imagens nas denúncias para facilitar a identificação do problema.

- Listagem de locais de descarte
Exibe pontos apropriados para descarte de diferentes tipos de resíduos.

- Persistência em banco de dados
As informações cadastradas ficam armazenadas para consulta e gerenciamento.

- Estrutura organizada de pastas
Projeto estruturado para facilitar manutenção e evolução.

# Tecnologias Utilizadas

HTML, CSS e JavaScript
